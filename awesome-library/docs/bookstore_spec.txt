Library Management Webapp

This is a webapp that helps manage library services.

A Library contains books ranging by categories.
Anyone can browse categories and books infos in the webapp.
The webapp allows visitors to sign up on the webapp and request
for borrowing books.

1st Iteration
	User stories: Subscription
    1. Subscription to awesome library.
      a. A visitor must be able to subscribe and must receive an email 
         including a confirmation link.
      b. A subscriber can log in and see his name throughout 
         the application after.
